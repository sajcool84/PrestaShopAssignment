PrestaShop - Automation
===========================
Tools used / Frameword:
	1> Maven project, TestNG, Selenium Webdriver
	2> Page object Model framework without Data driven

Scope :
	1> Accessing the website and clicking on SignIn
	2> New Registration - Create Account
	3> Registered User - SignIn

Out of / Not in Scope :
	1: Product purchase
	2: Datadriver by using excell sheet is not covered.
